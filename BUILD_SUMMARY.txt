🚀 QUICK START
═════════════════════════════════════════════════════════════════

1. INSTALL FFMPEG:
   Linux:    sudo apt-get install ffmpeg
   macOS:    brew install ffmpeg
   Windows:  choco install ffmpeg

2. INSTALL PYTHON DEPENDENCIES:
   pip install -r requirements.txt

3. TEST INSTALLATION:
   python video_compressor.py presets

4. COMPRESS YOUR FIRST VIDEO:
   python video_compressor.py compress input.mp4 output.mp4

5. TRY DIFFERENT QUALITIES:
   python video_compressor.py compress input.mp4 output_high.mp4 --quality high
   python video_compressor.py compress input.mp4 output_med.mp4 --quality medium

6. BATCH PROCESS:
   python video_compressor.py batch ./videos ./compressed

7. GET VIDEO INFO:
   python video_compressor.py info input.mp4


💻 CLI INTERFACE
═════════════════════════════════════════════════════════════════

MAIN COMMANDS:
  compress    Compress a single video
  batch       Batch process videos in directory
  info        Get detailed video information
  presets     Show available quality presets
  --help      Show help and options

COMPRESS OPTIONS:
  --quality   {ultra_high, high, balanced, medium, low}
  --audio     {aac, mp3, opus, copy}
  --keep-original  Keep or delete original file

EXAMPLES:
  python video_compressor.py compress video.mp4 out.mp4
  python video_compressor.py compress video.mp4 out.mp4 --quality high
  python video_compressor.py batch ./videos ./output
  python video_compressor.py info video.mp4


🐍 PYTHON API
═════════════════════════════════════════════════════════════════

from video_compressor import VideoCompressor

compressor = VideoCompressor()

# Compress single video
compressor.compress_video('input.mp4', 'output.mp4', quality='high')

# Get video information
info = compressor.get_video_info('video.mp4')

# Get file size
size_bytes, size_str = compressor.get_file_size('video.mp4')

# Batch compress
compressor.batch_compress('./videos', './output', quality='balanced')
